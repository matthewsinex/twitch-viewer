Twitch and Speedrun Utilities

Users
Searches Twitch users by user name or user id. Returns the profile picture, user name, user id, and view count.
Follows
Given a Twitch user id, gets that user's follows, then displays which of those channels are live. Also generates links for MultiTwitch and chat popouts.
Leaderboards
Returns leaderboards by series on Speedrun.com.
Speedruns
Returns all runs for a game, not just current leaderboards. Also displays World Record progression tables and charts.
